# Spring Web Client

## 1. 설명
Spring Web Client 예제이다. 포트는 8080을 사용하고 http://localhost:8081 API를 호출 한다.

## 2. 개발환경

* OpenJDK 17

* spring-boot 2.7.5

* spring-boot-starter-webflux

## 3. API 실행

### 1) Mono

* POST
  - http://localhost:8080/web/mono

### 2) Flux

* GET
  - http://localhost:8080/web/flux
